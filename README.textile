h1. Poll for phpWebSite

"Poll":http://github.com/meisteg/poll is a module for the "phpWebSite":http://phpwebsite.appstate.edu CMS that lets you place polls on your site. Polls can be placed on any content page or embedded into content blocks.

h2. Getting Started

* "Installation Instructions":http://wiki.github.com/meisteg/poll/install for this module.
* "Submit an issue":http://github.com/meisteg/poll/issues if you have questions or to report a problem.
